# 심각도 기준 가이드 (P0~P3)

> SKILL.md에서 이동된 P0~P3 등급 판정 기준 및 리포트 출력 형식 상세 가이드

---

## 심각도 등급

| 등급 | 의미 | 영향 |
|------|------|------|
| **P0** | 즉시 수정 | 검색 결과에 아예 안 나오거나, 모바일 검색 불가 |
| **P1** | 빠른 수정 | 검색 순위 하락, SNS 공유 미리보기 깨짐 |
| **P2** | 개선 권장 | 최적화 기회 누락, 사용자 경험 저하 |
| **P3** | 최적화 제안 | 미세한 개선, 경쟁사 대비 우위 확보 |

## 영역별 P0/P1 판정 기준

### P0 즉시 수정 (검색 결과 노출 불가 수준)

| 영역 | 이슈 | 이유 |
|------|------|------|
| robots.txt | `Disallow: /` (전체 차단) | 검색 결과에 아예 안 나옴 |
| robots.txt | 파일 없음 | 크롤러가 기본 정책 사용, 의도치 않은 차단 가능 |
| Sitemap | 파일 없음 | 크롤러가 페이지를 누락할 수 있음 |
| 메타태그 | `<title>` 없음 | 검색 결과 제목 표시 불가 |
| 메타태그 | `viewport` 없음 | 모바일 검색 순위 직격 |
| AI 크롤러 | GPTBot/ClaudeBot 차단 | AI 검색(Perplexity, ChatGPT 검색)에서 완전 제외 |

### P1 빠른 수정 (검색 순위 하락 수준)

| 영역 | 이슈 | 이유 |
|------|------|------|
| robots.txt | `Sitemap:` 지시자 없음 | 크롤러가 sitemap을 못 찾음 |
| Sitemap | 공개 라우트 누락 | 크롤러가 페이지를 누락 |
| 메타태그 | 중복 title | 페이지 구분 불가 |
| 메타태그 | description 없음 | 검색 스니펫 자동 생성 (품질 저하) |
| 메타태그 | og:image 없음 | SNS 공유 미리보기 깨짐 |
| 메타태그 | canonical 없음 | 중복 콘텐츠 페널티 위험 |
| 메타태그 | `lang` 없음 | 언어 타겟팅 실패 |
| 이미지 | alt 없음 | 접근성 + 이미지 검색 순위 |
| 성능 | CSR 전용 (SSR/SSG 없음) | 검색 봇이 콘텐츠 못 읽음 |
| 성능 | 웹폰트 display:swap 없음 | FOIT (텍스트 안 보임) → CLS 영향 |
| AI 크롤러 | CSR 전용 | AI 크롤러가 콘텐츠 수집 불가 |
| AI 크롤러 | x-robots-tag: noai 과다 | 의도치 않은 AI 차단 |
| AEO | FAQPage JSON-LD 없음 | Featured Snippet 기회 상실 |
| AEO | 질문형 헤딩 없음 | AI가 Q&A 패턴 인식 못함 |
| GEO | 수치/통계 없음 | AI가 인용 안 함 |
| GEO | 출처 표기 없음 | 신뢰도 낮은 소스로 판단 |
| GEO | 페이지 상단 핵심 답변 없음 | AI가 페이지 상단 우선 평가 |
| GEO | dateModified 없음 | AI가 최신성 판단 불가 |

## AEO/GEO 심각도 참고

> AI 검색에서의 가시성은 빠르게 중요해지고 있습니다.
> Gartner 예측에 따르면 2026년까지 유기 검색 트래픽의 25%가 AI 챗봇으로 이동합니다.
> AI Overview가 있는 쿼리의 유기 CTR은 61% 감소했습니다 (1.76% → 0.61%).

AEO/GEO 점수가 낮은 경우 SEO 기술 점수가 높더라도 AI 검색에서의 가시성이 크게 낮을 수 있습니다. 두 점수를 함께 보고 액션 플랜을 수립하세요.
